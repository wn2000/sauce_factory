The folder structure of this recipe is like this:

```
area51/
├── boxart
│   ├── addon.z.png
│   └── boxart.png
├── cartridge.xml
├── emu
│   └── mame2003_plus_libretro.so
├── exec.sh
└── roms
    ├── area51mx
    │   └── area51mx.chd
    ├── area51mx.zip
    └── hiscore.dat
```

Please obtain the rom `area51mx.zip` and disk image `area51mx.chd`
(compatible with the mame2003-plus libretro core) and place them
accordingly.
